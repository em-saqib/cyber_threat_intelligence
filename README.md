# Cyber Threat Intelligence (CTI) Learning Repository

This repository is created for learning and experimenting with Cyber Threat Intelligence (CTI) using modern machine learning techniques.

It focuses on:
- Named Entity Recognition (NER) for extracting threat-related entities (e.g., malware, IPs, vulnerabilities)
- Multi-class classification for identifying attack types and threat categories
- Applying transformer-based models such as BERT to real-world CTI datasets

The goal is to build practical understanding of how unstructured threat reports can be transformed into structured, actionable intelligence.

This repo includes experiments, code, and notes for research and hands-on learning in CTI and cybersecurity analytics.
